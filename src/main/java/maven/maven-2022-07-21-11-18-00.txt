Какие функции есть у сборщика Maven?

возможность управления зависимостями (dependency management)

возможность запускать разные задачи (build lifecycle goals)

возможность запускать компонентные тесты (unit tests)





Какой команды не существует в Maven?

mvn build

Maven использует build lifecycle
в котором можно указать что именно сделать с проектом

Такой общей задачи как build там нет,
но есть ряд более узких: verify, package, deploy
