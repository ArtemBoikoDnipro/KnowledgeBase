build tool - сборщик

.iml файл - файл идеи

чтобы пользоваться проектом, нам нужен сборщик, который позволит открывать проекты в разных средах разработки

проблема 1 - собранные через IDE проекты выглядят по разному в разных IDE

проблема 2 - принуждение всех на проекте пользоваться одной IDE

проблема 3 - отсутствие адекватной поддержки командной строки внутри IDE

проблема 4 - dependency hell, все библиотеки нужно загрузить и прилинковать к проекту

сборщик решает все эти проблемы
